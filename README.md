# gdevs server website

A GitHub Pages-ready website for `gdevsserver.minehut.gg`.

## Files

- `index.html` - homepage with rotating Minecraft-style panorama, join info, copy IP button, live status, and feature cards.
- `rules.html` - rules page.
- `docs.html` - command docs/wiki with bounties, Brewery, economy, and Advanced Fishing sections.
- `staff.html` - staff page.
- `404.html` - themed error page.
- `style.css` - full site styling.
- `script.js` - copy IP, live status, audio toggle, mobile nav.
- `assets/` - panorama screenshots, generated audio, and favicon.

## GitHub Pages setup

1. Upload all files to a GitHub repository.
2. Go to repository Settings.
3. Open Pages.
4. Set source to your main branch and root folder.
5. Save and wait for GitHub Pages to publish.

Do not move the `assets` folder unless you also update the paths in `style.css` and `script.js`.
